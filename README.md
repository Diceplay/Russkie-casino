<blockquote><p>При регистрации игрок может увидеть поле для ввода промо кода. При наличии секретной комбинации нужно указать ее для получения начисления.</p></blockquote>
<p>Далее придется заполнить подробную анкету. Операторы требуют указывать личную и контактную информацию. Сведения должны быть достоверными.</p>
<h3>Прохождение верификации</h3>
<p>Для ускорения дальнейших выплат стоит пройти верификацию. Она предполагает подтверждение личности. Игрок должен предоставить документ, который удостоверяет указанные в анкете данные. Подойдет паспорт, загранпаспорт или водительские права. Иногда требуют предоставить подтверждение адреса, а также платежных данных. Для последнего применяется фото банковской карты. Подойдет и скрин профиля в электронной платежной системе. Снимки или сканы загружаются в Личном кабинете или отправляются в саппорт.</p>
